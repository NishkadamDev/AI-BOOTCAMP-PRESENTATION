04 / MODEL WAR ANALYSIS

Claude vs Gemini

Claude Sonnet
◆ MY CODE ENGINE
  • Primary tool for all coding — React components, Python scripts, debugging
  • Caught logic errors and explained why something was wrong, not just what to fix
  • Held multi-step context well — could say "now add XP logic to that component" and it remembered
  • More honest about uncertainty — said "I'm not sure" instead of confidently hallucinating
  • Used for: every line of code in this bootcamp
Gemini 1.5 Pro
◆ MY EVERYTHING ELSE ENGINE
  • Research, brainstorming, planning — fast and wide coverage of any topic
  • Writing first drafts: project descriptions, README content, explanations
  • Synthesising multiple sources into a clear summary in seconds
  • Google ecosystem integration — Docs, Search, and Workspace felt seamless
  • Used for: ideation, writing, research, and anything non-code

Two AIs, two jobs. Claude wrote the code. Gemini handled everything else. Neither alone would have been enough.
